# hello-open-source-lab

A beginner-friendly lab for learning open-source contribution workflows — fork, branch, commit, PR.

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/LianderTobias/hello-open-source-lab.git
   cd hello-open-source-lab
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the project**
   ```bash
   npm start
   ```

## Contributing

Contributions are welcome! See the [issue tracker](https://github.com/LianderTobias/hello-open-source-lab/issues) for open tasks.

1. Fork the repo
2. Create your feature branch (`git checkout -b feature/my-change`)
3. Commit your changes (`git commit -m 'Add my change'`)
4. Push to the branch (`git push origin feature/my-change`)
5. Open a Pull Request
